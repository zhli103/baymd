# 就医选择与医院推荐指南

## 一、医院等级划分

### 三级甲等（三甲）
- **定义**：中国医院最高等级。具备全面医疗、教学、科研能力，综合实力最强。
- **特点**：科室齐全、设备先进、专家资源丰富。但患者量大、挂号难、排队时间长。
- **适合场景**：疑难重症、复杂手术、罕见病诊治、重大抢救。

### 三级乙等（三乙）
- **定义**：三级医院中稍逊于三甲，但也具备较强的综合实力。
- **适合场景**：中重度疾病、常规手术、专科治疗。性价比通常高于三甲。

### 二级甲等（二甲）
- **定义**：区县级医院的主力，承担区域内常见病多发病诊治。
- **特点**：挂号相对容易、费用低、服务接地气。
- **适合场景**：常见病、普通手术、康复治疗、慢性病管理。

### 社区卫生服务中心
- **特点**：方便快捷、费用低、家庭医生签约服务。
- **适合场景**：小病小痛、感冒发烧、血压血糖监测、疫苗接种、健康咨询。

## 二、如何按病情选择医院

### 初诊建议
1. **小病先去社区或二甲医院**：常见病基层医院完全能处理，没必要挤三甲。
2. **需要进一步检查时逐级转诊**：首诊医生判断需要上级医院时会建议转诊方向。
3. **急重症直接去综合三甲或专科医院**：心梗、脑卒中、严重外伤等直奔有能力处理的医院。

### 常见疾病医院选择参考

#### 心血管疾病
- **首选三甲心血管专科或有心内科优势的综合医院**
- 冠心病、心梗、心律失常 → 心内科
- 心脏搭桥、瓣膜置换 → 心外科
- 关键指标：医院年心脏介入手术量、是否有胸痛中心。

#### 肿瘤疾病
- **首选肿瘤专科医院或三甲肿瘤中心**
- 初诊建议综合医院的肿瘤内科/外科，确诊后可考虑专科医院。
- 关键指标：肿瘤科是否为国家/省级重点专科、是否有MDT多学科会诊制度。

#### 神经系统疾病
- 脑卒中 → 就近选择有卒中中心的三甲综合医院（时间就是大脑）。
- 癫痫、帕金森 → 神经内科专科。
- 脑肿瘤、脑出血手术 → 神经外科（需评估医院年手术量）。

#### 骨科
- 骨折、关节置换 → 三甲骨科或骨科专科医院。
- 运动损伤 → 运动医学科（康复与手术结合）。
- 脊柱疾病 → 脊柱外科专长医院。

#### 儿科
- 儿童普通疾病 → 二甲以上综合医院儿科或儿童医院。
- 儿童重症、新生儿疾病 → 儿童专科医院（设备更全）。
- 儿童外科手术 → 儿童医院外科（需有小儿麻醉能力）。

#### 妇产科
- 常规产检 → 二甲以上医院妇产科。
- 高危妊娠 → 三甲产科或妇产专科医院（具备危重孕产妇救治中心）。
- 妇科肿瘤 → 肿瘤医院妇科或三甲妇科。

## 三、就诊实用指南

### 挂号方式
1. **线上挂号**：医院官方APP/微信公众号/小程序、各省市统一预约平台（如北京114）。三甲医院专家号通常提前1-2周放号，热门专家号源当天秒光。
2. **现场挂号**：适合基层医院或非热门科室。部分号源保留给现场。
3. **加号**：已在本院就诊过的老患者可找医生申请加号，但不保证成功。

### 就诊准备（重要！）
1. **带齐证件**：身份证、社保卡/医保卡。
2. **带齐既往病历资料**：包括但不限于出院小结、手术记录、病理报告、影像资料和报告（CT/MRI/X光）、检验报告。
3. **列好问题清单**：提前写下想问医生的问题，避免面诊时遗忘。
4. **记录关键信息**：症状开始时间、变化过程、用药情况、过敏史。

### 医保使用
- **本地就医**：持社保卡在定点医院直接结算。
- **异地就医**：需先在国家医保服务平台APP或"国家异地就医备案"小程序办理备案。备案后可在就医地直接结算（报销比例可能低于本地）。急诊住院可先就医后补备案。
- **自费项目**：部分检查、药品和材料可能不在医保目录，就诊前可咨询。

### 住院准备
- 身份证、社保卡、住院押金。
- 个人生活用品（拖鞋、洗漱用品、水杯）。
- 既往病历资料。
- 家属联系方式，入院签署相关文书需要家属在场。

## 四、特殊就医场景

### 急诊（120急救）
- **什么是需要去急诊的情况**：突发剧烈胸痛、卒中症状（口眼歪斜、肢体无力）、大出血、严重呼吸困难、意识丧失、重度外伤。
- **拨打120时请说清楚**：具体地址、患者年龄性别、主要症状/伤情、是否有意识、呼吸情况。
- **到达急诊后**：分诊护士按病情危重程度分级，重症优先。急诊不急很正常（说明你不严重，是好事）。

### 第二诊疗意见
- **什么时候需要第二意见**：重大手术决策前、罕见病诊断、治疗不理想、需要确认诊断。
- **如何获取**：携带全套资料去同领域的另一家三甲医院挂专家号咨询。

## 五、重要提示
1. **急重症就近就医原则**：心梗脑卒中这类争分夺秒的病，先到就近有能力救治的医院，不要舍近求远。
2. **相信首诊医生判断**：医生比你更清楚疾病该看哪个科、去哪治。
3. **不盲信网络搜索**：网上信息鱼龙混杂，具体治疗方案一定要听医生的。
4. **医院选择不仅看名气，更要看科室实力**：某三甲综合排名高不一定其所有科室都强。了解该科室是否为国家级/省级重点专科更关键。
